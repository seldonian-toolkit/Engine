seldonian.RL.Agents.Policies.Softmax.MixedSoftmax¶

class MixedSoftmax(hyperparam_and_setting_dict, env_description, alpha=0.5)¶

Bases: Softmax

__init__(hyperparam_and_setting_dict, env_description, alpha=0.5)¶

Softmax mixed policy, as in https://people.cs.umass.edu/~pthomas/papers/Thomas2015b.pdf see equation for mu in last paragraph before Section 2.1. Stationarity

Parameters:

  • hyperparameter_and_setting_dict – Specifies the environment, agent, number of episodes per trial, and number of trials

  • env_description (Env_Description) – an object for accessing attributes of the environment

  • alpha – The mixing hyperparameter. 100%alpha is how far from behavior policy the mixed policy can be.
