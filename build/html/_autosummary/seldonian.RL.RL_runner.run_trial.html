seldonian.RL.RL_runner.run_trial¶

run_trial(hyperparameter_and_setting_dict, model_params=None, parallel=False, n_workers=8)¶

Run a single trial consists of an arbitrary number of episodes.

Parameters:

hyperparameter_and_setting_dict (dict) – Specifies the environment, agent and number of episodes to run

Model_params:

Policy parameters to set before running the trial

Parallel:

Whether to use parallel processing

N_workers:

Number of cpus if using parallel processing

Returns:

(List of episodes, agent)
