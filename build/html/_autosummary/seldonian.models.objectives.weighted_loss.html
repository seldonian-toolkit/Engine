seldonian.models.objectives.weighted_loss¶

weighted_loss(model, theta, X, Y)¶

Calculate the averaged weighted cost: sum_i p_(wrong answer for point I) * c_i where c_i is 1 for false positives and 5 for false negatives

Parameters:

  • model – SeldonianModel instance

  • theta (numpy ndarray) – The parameter weights

  • X (numpy ndarray) – The features

  • Y (numpy ndarray) – The labels

Returns:

weighted loss such that false negatives have 5 times the cost as false positives

Return type:

float
