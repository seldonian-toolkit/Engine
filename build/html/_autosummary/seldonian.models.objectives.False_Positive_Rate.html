seldonian.models.objectives.False_Positive_Rate¶

False_Positive_Rate(model, theta, X, Y, **kwargs)¶

Calculate probabilistic mean false positive rate over the whole sample. This has slightly different meanings depending on binary vs. multi-class setting.

Parameters:

  • model – SeldonianModel instance

  • theta (numpy ndarray) – The parameter weights

  • X (numpy ndarray) – The features

  • Y (numpy ndarray) – The labels

Returns:

Average false positive rate

Return type:

float between 0 and 1
